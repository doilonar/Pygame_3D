# Pygame_3D
## What is this?
Is a 3D game make in Python with Pygame.
## What is your mission?
Is to kill all the npcs. The map is generated random with numpy. Have fun!!!
## How to run the game?
```
python3 main.py
```
## Packets you need
 ```
 pip3 install numpy
 
 ```
 ```
 pip 3 install pygame
 ```
